#  Site de Futebol

Este é um projeto de um site de futebol desenvolvido por mim como forma de prática de HTML e CSS.

##  Sobre o projeto
Criei este site para treinar a estruturação de páginas web e estilização com CSS. O projeto simula um site de futebol com layout simples e organizado.

## Tecnologias usadas
- HTML5
- CSS3

## 📚 O que aprendi
- Estruturação com HTML
- Estilização com CSS (cores, layout, espaçamento)

## ⚠️ Observação
Este projeto ainda não é responsivo. Pretendo adicionar responsividade em versões futuras.

## 🎯 Objetivo
Evoluir minhas habilidades como desenvolvedor front-end e melhorar meus projetos com o tempo.
